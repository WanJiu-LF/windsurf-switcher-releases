## ✨ 核心功能

### 🎯 多账号管理
- 一键添加无限账号（账号 + 密码 + 备注）
- 隐藏式浏览器自动登录，全程无需你动手
- 账号信息加密本地存储，跨机器迁移用一份加密备份文件搞定
- 一键导出 / 导入 `.wsb` 备份

### ⚡ 无感切换
- 切换账号**不需重启 Windsurf**，新账号瞬间生效
- IDE 状态全保留：编辑窗口、终端会话、未保存的改动 —— 通通不丢
- 切完即用，体感 < 1 秒

### 📊 智能监控 & 自动换号
- 实时查询每个账号的 Cascade 额度（Free / Pro 套餐自适应）
- 设阈值（如 < 20% 自动换号）→ 后台守护监控 → 不足自动切下一个有额度的账号
- 详细切换历史 / 错误日志，每一次动作都可追溯

### 📈 使用画像
- Cascade 写过的代码行数、模型用量分布、对话总数
- 按日 / 周 / 月聚合，热力图可视化使用习惯
- 多账号合计，一眼看清你的 AI 编程总投入

### 🚀 极致体验
- 现代 UI，深浅主题随系统切换
- 系统托盘后台运行，关窗不退出
- 开机自启 + 启动时直接到后台
- 安装版支持自动更新，无感升级到最新版本
